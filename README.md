# SEQUENTIAL AND PARALLEL MERGE SORT

This repo contains solutions and benchmark tests of "SEQUENTIAL AND PARALLEL MERGE SORT" on Golang

## HOW 2TEST

1. Update dependencies

```
$ make build
```

2. Run tests

```
$ make test
```

## HOW 2RUN

1. Update dependencies

```
$ make build
```

2. Run script
```
$ make run
```
